# 易房拼拼 · Brand Spec

来源：用户提供的官方 logo（红底 + 金黄房屋图形 + 「易房拼拼」字标 + 「让买房更简单」slogan）。
> ⚠️ Logo 原图为聊天粘贴，未落盘。当前 HTML 用「金色房屋图标 + 易房拼拼字标」做的 on-brand 临时锁版，待用户提供 logo PNG/SVG 后替换 `.brand-mark` 内的图标。

## 色板（从 logo 提取 + 微调为可用系统）
| 角色 | 色值 | 用途 |
|------|------|------|
| 品牌红（主） | `#D8323B` | 主色，hero band、强调、按钮 |
| 品牌红（深） | `#A8242C` | 深色变体，渐变底、hover |
| 品牌金（主） | `#F2B43A` | 笋度、高亮、金色细线、次按钮 |
| 品牌金（亮） | `#FFD466` | 高光、徽章 |
| 暖象牙底 | `#FBF6EC` | 页面基底（不用纯白，避免冷感） |
| 纸面卡片 | `#FFFFFF` | 卡片 |
| 墨色文字 | `#211C18` | 正文主色 |
| 次级文字 | `#6B5F54` | 辅助文字（暖灰） |
| 描边 | `#EADFCB` | 暖色分隔线 |

## 字体
- 中文：系统中文栈优先（China-friendly，不依赖 Google Fonts）
  `"PingFang SC","HarmonyOS Sans SC","Microsoft YaHei","Source Han Sans SC",sans-serif`
- 衬线（大标题/数字点缀）：`"Songti SC","STSong","Noto Serif SC",serif` —— 仅用于 hero 数字与英文，营造质感
- 不用 Inter/Roboto 作 display

## 气质关键词
靠谱 · 透明 · 专业 · 大湾区 · 喜庆而不土 · 数据感（笋度）

## 反 slop 约束
- 不用紫渐变、不用 emoji 当图标、不堆装饰 icon
- 金色作为「细线 + 笋度徽章」点缀，不大面积平铺（避免红黄土味）
- 红色为主导品牌色，金色克制使用
- 房源数据用真实 11 套（Supabase），不编造

## 定位文案（用户给定 + 生成）
- Slogan：立足深圳 · 放眼大湾区
- 使命：让买房更简单
- 一句话定位：做让用户放心靠谱的笋盘代理人
- 双业务：① 笋盘匹配（笋度评分体系）② 房产金融贷款服务
